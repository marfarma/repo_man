application programming interface

Repo Man has an API that is accessible via Active Resource. For ease of use, there is a gem that creates a thin wrapper around Active Resource, thus:


>> require 'rubygems'
=> true
>> require 'repo_man'
=> true
>> food = RepoMan::Repository.create(:name => 'Food (Meat Flavored)', :scm => 'git')
=> #<RepoMan::Repository:0x17c2ecc @attributes={"name"=>"Food (Meat Flavored)", "updated_at"=>Tue Apr 21 17:26:42 UTC 2009, "id"=>18, "scm"=>"git", "path"=>"git.lab.viget.com:/srv/git/food-meat-flavored.git", "created_at"=>Tue Apr 21 17:26:42 UTC 2009}, prefix_options{}
>> RepoMan::Repository.find(food.id).name
=> "Food (Meat Flavored)"
>> food.destroy
=> #<Net::HTTPOK 200 OK readbody=true>
